## ✨ 核心功能

### 1. 📸 照片预览和下载

**功能描述**
- 在 StoryboardNode 中显示高清照片预览
- 支持照片翻页导航（上一张/下一张）
- 显示照片计数（如 1/3）
- 单张照片下载
- 批量下载所有照片

**实现细节**
- 使用 React 状态 `activeImageIndex` 管理当前图片
- 导航按钮在首/末图片时自动禁用
- 下载使用 `<a>` 标签 + blob URL 机制
- 样式响应式设计，适配不同屏幕

### 2. 🎬 视频预览和下载

**功能描述**
- 在 VideoNode 中在线播放视频
- 支持视频翻页导航
- 显示视频计数
- 单个视频下载
- 批量下载所有视频
- 实时进度条显示

**实现细节**
- 使用原生 `<video>` 标签播放
- 支持播放器控制（播放/暂停/音量等）
- 进度条实时更新（3 秒轮询）
- 状态徽章显示生成状态

### 3. 🔗 链式操作（照片→视频）

**工作流程**
```
用户点击"生成视频" 
  ↓ 触发自定义事件
ProjectWorkspace 监听事件
  ↓ 更新 VideoNode 为"生成中"
调用后端 /video/generate API
  ↓ 轮询 /video/task/{taskId}
显示实时进度
  ↓ 生成完成
自动更新 VideoNode 数据
  ↓ 保存 Canvas 布局
```

**技术亮点**
- 使用 `CustomEvent` 跨组件通信
- 函数式状态更新避免闭包陷阱
- Promise 链 + async/await 管理异步
- 自动错误处理和状态回退

## ⚠️ 已知限制

1. **任务存储**
   - 当前使用内存存储
   - 服务重启后任务信息丢失
   - 建议：迁移到 Redis 或数据库

2. **并发处理**
   - 同时生成多个视频可能造成限流
   - 建议：实现队列管理

3. **文件大小**
   - 大文件下载可能超时
   - 建议：使用分片下载

4. **实时更新**
   - 使用轮询而非 WebSocket
   - 建议：升级为 WebSocket
